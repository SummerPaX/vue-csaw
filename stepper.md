# Aufgabe - Stepper

1. Der Stepper besteht aus  
    - decrement Button
    - output Value
    - increment Button
2. The buttons should increment/decrement the output value
3. The value should not go below 0
4. The output value should have different colors depending on its value
    - 0 black
    - 1-5 red
    - 6-10 amber
    - 11-15 yellow
    - 16-20 green
